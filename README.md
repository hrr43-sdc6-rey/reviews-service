| ENDPOINTS      |  TYPE   |   Operation                           |
|  ------------  | ------- |  ----------------------------------   |
| /reviews/:id   |  GET    |  retrieves a review                   |
| /reviews/      |  POST   |  adds a review                        |
| /reviews/:id   |  PUT    |  updates/replaces a field in a review |
| /reviews/:id   |  DELETE |  removes a review                     |
